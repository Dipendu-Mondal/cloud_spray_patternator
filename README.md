
# Cloud Spray Patternator

A cloud-based spray analysis system inspired by the uploaded research paper.

## Features
- Spray angle detection
- Spray distribution analysis
- Peak / solid stream jet detection
- Cloud deployment ready

## Tech Stack
- Streamlit
- OpenCV
- SciPy
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy Free

### Streamlit Cloud
1. Upload files to GitHub
2. Go to https://share.streamlit.io
3. Deploy app
